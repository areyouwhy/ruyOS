# Resources

Reusable building blocks — prompts, frameworks, examples, and snippets. Think of this as a library that skills and agents can pull from.

## Subfolders

- **Prompts/** — Proven prompts for recurring tasks
- **Frameworks/** — Mental models, decision frameworks, analysis templates
- **Examples/** — Good output examples (posts, emails, presentations) for AI reference
- **Snippets/** — Reusable code snippets, text blocks, email templates

## How to use

When building a skill, point it to resources here instead of embedding copies. When you produce something good, save an example here for future reference.
